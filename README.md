> 内部组件库文档, 包含App端和Web端两套
>
> App技术栈为Flutter+Material, Web技术栈为Vue+ElementUI
>

## 如何启动
全局安装 [docsify-cli](https://docsify.js.org/#/zh-cn/quickstart) 工具

```
docsify serve
```
